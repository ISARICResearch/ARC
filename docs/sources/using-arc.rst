Using ARC
=========

You can use ARC for research or study in different ways:

- Direct Access: You can download the latest ARC versions as `GitHub releases <https://github.com/ISARICResearch/ARC/releases>`_.

- Integration with BRIDGE: ARC is fully integrated into the `BRIDGE <https://bridge.isaric.org>`_ software tool for CRF generation.

- Contributions: While contributions to the ARC schema and related documents are limited to authorized users, you can create GitHub `issues <https://github.com/ISARICResearch/ARC/issues>`_ or `discussions <https://github.com/ISARICResearch/ARC/discussions>`_ for raising questions, suggestions, or clarifications.
